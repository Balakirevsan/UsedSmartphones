# Full-Stack Web Application

Modern web application built with React frontend and FastAPI backend, using MongoDB Cloud as database.

## Technology Stack

### Frontend
- React
- React Router
- Axios
- Material-UI/Tailwind CSS
- Redux/Context API

### Backend
- FastAPI
- MongoDB Atlas
- PyMongo
- Python-dotenv
- JWT Authentication
